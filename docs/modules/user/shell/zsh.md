# Модуль `user/shell/zsh`

**Путь:** `modules/user/shell/zsh.nix` 
**Назначение:** Настройки zsh для кроссплатформенных пользователей.

## Описание
(текст описания модуля)

## Опции
| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|-----------|
| `dotfiles` | bool | true | Подключить пользовательские алиасы из домашней директории `$HOME/.zshrc.d`. |
| `aliases` | list of strings | [] | Дополнительные глобальные алиасы, добавляемые в конец конфигурации. |
| `pathAdd` | list of paths | [] | Путь к внешним исполняемым скриптам (например `$HOME/.local/bin`). |

### Пример использования
```
{ pkgs, ... }: {
  imports = [ ./user/shell/zsh.nix ];
}
```
