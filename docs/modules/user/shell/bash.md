## Опции

|----------|-----|--------------|-----------|
| `dotfiles` | bool | true | Подключить пользовательские настройки из `$HOME/.bashrc.d`. |
| `aliases` | list of strings | [] | Глобальные алиасы, добавляемые в конце `.bashrc`. |
| `pathAdd` | list of paths | [] | Путь к внешним скриптам.|
