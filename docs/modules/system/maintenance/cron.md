# Модуль `system/maintenance/cron`

**Путь:** `modules/system/maintenance/cron.nix`
**Назначение:** Включение планировщика cron-заданий.
**Используется хостами:** все NixOS-хосты

## Описание

Включает системный cron-демон для запуска запланированных задач.

## Настройки

```nix
services.cron.enable = true;
```

## Параметры

Модуль не экспонирует пользовательских опций.
