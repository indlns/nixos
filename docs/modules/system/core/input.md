# Модуль `system/core/input`

**Путь:** `modules/system/core/input.nix`
**Назначение:** Раскладка клавиатуры X11.
**Используется хостами:** все NixOS-хосты

## Описание

Устанавливает раскладку клавиатуры `us` (американская) для X11.

## Настройки

```nix
services.xserver.xkb = {
  layout = "us";
  variant = "";
};
```

## Параметры

Модуль не экспонирует пользовательских опций.
