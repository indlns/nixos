# Модуль `system/core/shell`

**Путь:** `modules/system/core/shell.nix`
**Назначение:** Включение оболочки zsh на системном уровне.
**Используется хостами:** все NixOS-хосты

## Описание

Включает zsh как системную оболочку. Пользовательская конфигурация zsh (плагины, тема) находится в `modules/user/shell/zsh.nix`.

## Настройки

```nix
programs.zsh.enable = true;
```

## Параметры

Модуль не экспонирует пользовательских опций.
