# Модуль `system/core/swap`

**Путь:** `modules/system/core/swap.nix`
**Назначение:** Конфигурация swap-пространства.
**Используется хостами:** все NixOS-хосты

## Описание

Создаёт swap-файл размером 8 ГБ в `/swapfile`.

## Настройки

```nix
swapDevices = [{
  device = "/swapfile";
  size = 8192; # 8GB
}];
```

## Параметры

Модуль не экспонирует пользовательских опций.
