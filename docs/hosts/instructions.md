# Введение в хост‑конфигурацию

Файл `hosts/<host>/configuration.nix` является точкой входа для каждой машины. Он импортирует флейк‑модуль и список системных модулей, затем задаёт специфические опции.

## Структура `configuration.nix`
```nix
{ config, pkgs, ... }: {
  imports = [
    ./hardware-configuration.nix            # autogenerated (не редактируем!)
    ../../modules/system/default.nix        # основной набор модулей
    ../path/to/host-specific-modules.nix     # собственные настройки хоста
  ];

  ... # остальные атрибуты NixOS: system.stateVersion, services, networking и т.д.
}
```

### Главное правило
Не редактировать `hardware-configuration.nix`. Все пользовательские опции помещаются только в файлы `configurations.*.nix` или в отдельные модули, подключаемые через `imports`.
