# Хост `ms-02`

**Тип:** NixOS
**Железо:** Intel Xeon E5-2680 v4, NVIDIA RTX 2080, Intel iGPU
**Назначение:** Основной физический сервер. Хранение данных (NFS), контейнеры (Docker), AI-серверы.

## Конфигурация

| Файл | Путь |
|------|------|
| Основной | `hosts/ms-02/configuration.nix` |
| Аппаратная | `hosts/ms-02/hardware-configuration.nix` |

## Системные настройки

- **Загрузчик:** systemd-boot (EFI)
- **Ядро:** параметр `i915.enable_guc=3` (для Intel iGPU)
- **State version:** 25.11
- **Доп. пакеты:** intel-gpu-tools, libva-utils, vulkan-loader, vulkan-tools, cudatoolkit

## Подключённые модули

| Модуль | Путь | Описание |
|--------|------|----------|
| Всё из `default.nix` | `modules/system/default.nix` | Базовый набор (core, networking, security, services) |
| graphics | `modules/system/hardware/graphics.nix` | OpenGL + 32-бит |
| igpu | `modules/system/hardware/igpu.nix` | Intel iGPU: VAAPI (iHD), Quick Sync, OpenCL |
| dgpu | `modules/system/hardware/dgpu.nix` | NVIDIA: драйвер, modesetting, container toolkit, nvidia-docker |
| nfs | `modules/system/services/network/nfs.nix` | NFS-монтирования с NAS |
| docker | `modules/system/virtualization/docker.nix` | Docker daemon |

## Сервисы

| Сервис | Порт | Описание |
|--------|------|----------|
| SSH | 22 | OpenSSH (root=no, password=no) |
| Docker | 2375 | Docker daemon (только локально) |
| NFS client | — | Монтирования с NAS 192.168.10.176 |
| Printing | 631 | CUPS |
| NVIDIA | — | GPU passthrough для контейнеров |

| Точка монтирования | Путь на NAS | Опции |

| `/mnt/nas/documents` | `/mnt/data/documents` | auto, nofail, noatime, nolock, intr, tcp |
| `/mnt/nas/photo` | `/mnt/data/photo` | auto, nofail, noatime, nolock, intr, tcp |
