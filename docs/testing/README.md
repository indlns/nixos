# Тестирование

## Структура тестов

```
tests/
├── unit/          # Проверка синтаксиса и опций
├── integration/   # Сборка модулей в песочнице
└── e2e/           # Полная проверка системы
```

## Типы тестов

| Тип | Описание | Команда запуска |
|-----|----------|-----------------|
| **Unit** | Проверка синтаксиса Nix, импортов, опций | `nix flake check .` |
| **Integration** | Сборка конкретного хоста в изолированном окружении | `sudo nixos-rebuild test --flake .#<host>` |
| **E2E** | Полная проверка после развёртывания | `tests/e2e/run-tests.sh` |

## Запуск локально

```bash
# Unit-тесты: проверка всех конфигураций
nix flake check .

# Integration: тестирование конкретного хоста
sudo nixos-rebuild test --flake .#ms-02
sudo nixos-rebuild test --flake .#n10-nixos

# E2E: полная проверка
cd tests/e2e && ./run-tests.sh
```

## Добавление нового теста

1. Создать файл в соответствующем каталоге (`tests/unit/`, `tests/integration/` или `tests/e2e/`)
2. Описать проверяемый сценарий
3. Убедиться, что тест проходит локально
4. CI автоматически подхватит новый тест
