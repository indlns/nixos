# Сервис Desktop

**Модуль:** `modules/system/services/desktop/desktop.nix`
**Назначение:** Настройки графического окружения.

## Текущее состояние

Все сервисы **закомментированы**: X11, KDE Plasma, Firefox, ввод.

## Модули

| Сервис | Путь модуля | Состояние |
|--------|-------------|-----------|
| Desktop | `services/desktop/desktop.nix` | Закомментирован |
| Firefox | `services/desktop/firefox.nix` | Закомментирован |
| Input | `services/desktop/input.nix` | Закомментирован |
