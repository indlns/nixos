# Сводка сервисов проекта

Таблица ниже содержит основные типы системных служб, упакованных в NixOS‑модули. Для каждого сервиса указано расположение модуля, краткое описание и ссылку на подробную документацию (README / DETAILS). Все сервисы находятся внутри каталога `modules/system/services/`.

| Тип | Путь к модулю | Краткое описания | Документация |
|-----|---------------|-------------------|--------------|
| Desktop | `services/desktop` | Настройки X11/Wayland, графического окружения и ввод‑вывода. | [Desktop Docs](../services/desktop) |
| AI | `services/ai` | Модели LLM (Llama‑CPP, Ollama) в виде systemd‑units. | [AI Docs](../services/ai) |
| Audio | `services/audio` | Подключение PipeWire и конфигурация аудио драйверов. | [Audio Docs](../services/audio) |
| Network | `services/network` | Опции NFS, VPN, DHCP и прочие сетевые компоненты. | [Network Docs](../services/network) |

> **Пояснение:** каждый модуль самодокументируется в своих директориях: `README.md`, `details.md`. При добавлении нового сервиса необходимо создать аналогичную структуру.
